
# Hello World in React

This is a simple project that demonstrates how to build a "Hello World" page using:

- ✅ Pure HTML
- ✅ JavaScript to manipulate the DOM
- ✅ React (via CDN)

## 💡 What You'll Learn

- How to create and render elements using React's `React.createElement`
- How to use `ReactDOM.createRoot()` and `root.render()`
- How to work with nested elements in React
- How to use a CDN to include React without installation

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. You’ll see a "Hello World" message created using React.

## 📦 Technologies Used

- HTML5
- JavaScript (ES6+)
- React 18 (via CDN)
- Git & GitHub

## 🌐 CDN Links Used

```html

<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js" crossorigin></script>
